Projects

Here are some of my current and past projects.

MemoryMosaic

MemoryMosaic is a really cool memory visualization tool. It transforms code repositories into artistic visual representations, helping developers "see" the evolution of their projects and codebase over time.

CodeBloom

CodeBloom is a collaborative gardening game for developers. Each code commit contributes to the growth of a virtual garden, where different types of code plants represent different programming languages and practices.

SynthVibe

SynthVibe is a collaborative music composition platform for developers. It uses code snippets to create unique soundscapes, allowing multiple developers to contribute to a shared musical masterpiece through coding.
